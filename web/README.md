# SecureHealth Login  
Baseline implementation of EIP-712 typed login (SIWE-style) with Next.js, wagmi, and iron-session.  

## Features  
- Wallet connect (Sepolia testnet)  
- EIP-712 typed login signing  
- Basic session handling  

## Getting Started  
```bash
pnpm install
pnpm dev
```
